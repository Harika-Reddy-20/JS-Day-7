Array Methods: Map,Filter,Reduce,Index,Push,Pop,Shift,Unshift,Slice,Some,Every

Array:

We can store any type of data type.

Index:

The 1st index of an array is strted with 0.

Syntax:

const value = arr.indexOf(3);

Array Methods:

Map:

Map is an method in js , which is used to transform an array ie., which will itearte through each and every element.

Syntax:

const result =arr.map( x => x*2);

Filter:

which checks the condition and filter all the elements in array and return you a new array.

Syntax:

const result1 = arr.filter(x=> x>3);

Reduce:

whenever the sum of all eements in an array or whenever we want to find the max value in array.

Syntax:

const total = arr.reduce((accumulator,current) => accumulator=accumulator+current,0);

Push:

It will add one more value at the en dof the array.

Syntax:

arr.push(7);

Pop:

It removes an element from the end of the array

Syntax:

arr.pop();

Shift:

It will remove first element an array.

Syntax:

arr.shift();

Unshift:

It will add one value at the first of an array.

Syntax:

arr.unshift(1);

Slice:

It will not manipulate the array, it will try to remove elements from an array.

Syntax:

const val3 = arr.slice(1,3);

Some and Every:

Both these are take in a condition and check if all the elements in th array is satisfying the condition or not , they always return a boolean value.

Syntax 1:

const val1 = arr.some(x => x%2 == 0);

Syntax 2:

const val1 = arr.every(x => x%2 == 0);
